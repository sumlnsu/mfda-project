# Analysis

이 디렉터리는 남성 패션 커뮤니티 데이터 분석 과정과 산출물을 보관합니다.

## Contents

- `notebooks/`: 분석 및 대시보드 데이터 생성 노트북
- `data/`: 키워드 후보, 정제 사전, 대시보드용 집계 데이터
- `plots/`: 분석 결과 이미지

## Role in Portfolio

대시보드가 보여주는 결과의 근거가 되는 분석 자산입니다. README와 면접 설명에서는 이 디렉터리를 통해 다음 흐름을 설명할 수 있습니다.

1. 커뮤니티 게시글 데이터를 분석 가능한 키워드 단위로 정리
2. 브랜드, 아이템, 스타일 키워드 사전 구성
3. 시즌별, 커뮤니티별 언급량과 비중 집계
4. Tableau 및 Streamlit 대시보드용 데이터 생성
