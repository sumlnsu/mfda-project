# MFDA Project Summary

MFDA는 남성 패션 커뮤니티 데이터를 활용해 브랜드, 아이템, 스타일 키워드의 변화와 커뮤니티별 차이를 분석한 프로젝트입니다.

## Problem

남성 패션 커뮤니티는 연령대, 취향, 스타일 선호에 따라 서로 다른 키워드 반응을 보입니다. 단순 언급량만 보면 특정 브랜드나 아이템이 왜 부상했는지, 어느 커뮤니티에서 강하게 반응했는지 설명하기 어렵습니다.

## Approach

- 커뮤니티 게시글에서 브랜드, 아이템, 스타일 키워드 후보를 정리
- 키워드 사전과 불용어를 관리해 분석 단위를 정제
- 연도, 시즌, 커뮤니티 기준으로 언급량과 비중을 집계
- 공출현 데이터를 활용해 키워드 간 관계를 탐색
- Tableau와 Streamlit을 통해 분석 결과를 대시보드로 제공

## Output

- 트렌드 분석 대시보드
- 커뮤니티 세그먼트 대시보드
- 키워드 탐색기
- LMC 브랜드 트렌드 분석 리포트

## DA Portfolio Message

이 프로젝트는 데이터 분석 직무 관점에서 다음 역량을 보여줍니다.

- 텍스트 데이터 정제와 분석 단위 설계
- 도메인 기반 키워드 분류
- 시즌별, 커뮤니티별 지표 설계
- 분석 결과의 대시보드화
- 브랜드 관점의 인사이트 리포팅
